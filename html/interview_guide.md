我已经为你生成了一个交互式的面试准备网页：[interview_prep.html](file:///e:/chain/hsa-agent/html/interview_prep.html)。

该工具集成了 50 道高频面试题（已精简核心 Q&A），采用了高端的暗黑模式设计，并支持以下功能：
- **折叠展示**：点击问题即可查看满分参考答案。

## 🌟 核心高分答案速览

### 1. 关于“SQL 幻觉”的硬核回答
**提问**：如何防止 LLM 在写 SQL 时编造字段？
**回答核心**：
- **物理铁幕策略**：在 Prompt 中动态注入 ClickHouse 的实时物理 Schema，剥夺 LLM 的“创造权”。
- **AST 强校验**：使用 SQLGuardian 模块对生成的 SQL 进行抽象语法树扫描，任何不在注册表中的字段都会触发物理隔离并强制熔断。

### 2. 关于“LangGraph 架构”的优越性
**提问**：为什么不用 LangChain？
**回答核心**：
- **循环纠错 (Self-Correction)**：审计场景需要多次重试（例如 SQL 报错后由 Critic 诊断并反馈给 Coder）。LangGraph 的循环图结构比线性的 Chain 更能处理这种复杂的闭环逻辑。

### 3. 关于“毫秒级性能”
**提问**：如何在高并发场景下保持响应？
**回答核心**：
- **语义路由 (Fast Route)**：通过向量相似度识别已知任务，直接跳过 LLM 走预定义的算法算子，将响应时间从秒级降至毫秒级。
- **语义 SQL 缓存**：缓存经过验证的“金牌 SQL”，实现零延迟复用。
