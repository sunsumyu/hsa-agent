## 6. 下一步：修复 Format 冲突（Fix-06）

`reporter_node` 当前逻辑：
```python
# 期待 LLM 返回 JSON
json_match = re.search(r'(\{.*\})', res_text, re.DOTALL)
report = AuditReport(**data)
```

但新 `REPORTER_PROMPT` 要求 LLM 输出五章节 Markdown，两者矛盾。

**解决方案**：让 reporter_node 接受两种输出格式：
1. 如果 LLM 返回了 JSON → 走原来的解析路径
2. 如果 LLM 返回了 Markdown（包含"### 四、核查结论"等章节）→ 提取内容生成 AuditReport，并把 Markdown 全文作为 content 发出

修复这个冲突后，预计得分将从 29 分跃升至 45+ 分。
